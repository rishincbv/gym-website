# Stitch Design Assets

Place exported Stitch files here for pixel-perfect UI conversion.

## Expected structure

```
design/stitch/
├── manifest.json       # Auto-generated by fetch script
└── raw/
    ├── landing.html
    ├── landing.png
    ├── login.html
    ├── login.png
    ├── register.html
    ├── register.png
    ├── dashboard.html
    ├── dashboard.png
    └── screen-5.html
```

## How to export from Stitch

1. Open your [Stitch project](https://stitch.withgoogle.com/projects/8510852350685465056)
2. Select a screen
3. Click **More** → **Export** → Download ZIP or copy code
4. Save HTML + screenshot to `raw/` using the names above

## Or use the fetch script

```bash
# Add STITCH_API_KEY to .env at project root
npm run fetch:stitch
```
